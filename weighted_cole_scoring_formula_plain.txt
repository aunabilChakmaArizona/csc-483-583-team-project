Weighted-Cole Scoring Formula - Plain

Main formula

final_score(doc) =

+ 0.0 * title_score
  index: whoosh_cole_index
  search in index: title field
  input used: transformed query text = clue, or category + clue if --include-category; then full text or entity-only text depending on query mode

+ 0.0 * redirect_score
  index: whoosh_cole_redirect_index
  search in index: redirect title field
  input used: transformed query text = clue, or category + clue if --include-category; then full text or entity-only text depending on query mode

+ 15.0 * body_score
  index: whoosh_cole_index
  search in index: full cleaned article body field
  input used: transformed query text = clue, or category + clue if --include-category; then full text or entity-only text depending on query mode

+ 0.0 * first_sentence_score
  index: whoosh_cole_first_sentence_index
  search in index: first sentence only, stored as body field
  input used: transformed query text = clue, or category + clue if --include-category; then full text or entity-only text depending on query mode

+ 0.0 * first_two_sentences_score
  index: whoosh_cole_first_two_sentences_index
  search in index: first two sentences only, stored as body field
  input used: transformed query text = clue, or category + clue if --include-category; then full text or entity-only text depending on query mode

+ 0.0 * first_paragraph_score
  index: whoosh_cole_first_paragraph_index
  search in index: first paragraph only, stored as body field
  input used: transformed query text = clue, or category + clue if --include-category; then full text or entity-only text depending on query mode

+ 0.0 * year_match_score
  index: whoosh_cole_index
  search in index: title field and body field, but only for extracted 4-digit years
  input used: years extracted from transformed query text

+ 0.0 * quote_match_score
  index: whoosh_cole_index
  search in index: body field, using tokens from quoted spans only
  input used: quoted text extracted from transformed query text

+ 0.0 * category_first_sentence_score
  index: whoosh_title_body_category_index
  search in index: title_first_sentence_categories keyword field
  input used: Jeopardy category only

+ 0.0 * category_first_two_sentences_score
  index: whoosh_title_body_category_index
  search in index: title_first_two_sentences_categories keyword field
  input used: Jeopardy category only

+ 1.0 * faiss_score
  indexes used inside this score: dpr_faiss/title_first_sentence, dpr_faiss/title_first_two_sentences, dpr_faiss/title_first_paragraph, dpr_faiss/title_entire_article
  search in indexes: DPR dense vectors for title + passage variant
  input used: one DPR query embedding from "The clue is: {clue}" or "The category is: {category}. The clue is: {clue}"

+ 1.0 * natural_questions_avg_faiss_score
  indexes used inside this score: dpr_faiss/title_first_sentence, dpr_faiss/title_first_two_sentences, dpr_faiss/title_first_paragraph, dpr_faiss/title_entire_article
  search in indexes: DPR dense vectors for title + passage variant
  input used: five DPR query embeddings from natural questions 1 to 5

+ 1.0 * natural_questions_concat_faiss_score
  indexes used inside this score: dpr_faiss/title_first_sentence, dpr_faiss/title_first_two_sentences, dpr_faiss/title_first_paragraph, dpr_faiss/title_entire_article
  search in indexes: DPR dense vectors for title + passage variant
  input used: one DPR query embedding from concatenated natural questions 1 to 5


FAISS score details

faiss_score
- sub-index: dpr_faiss/title_first_sentence
  searched content: title + first sentence dense vectors
  input: one default DPR query embedding
- sub-index: dpr_faiss/title_first_two_sentences
  searched content: title + first two sentences dense vectors
  input: one default DPR query embedding
- sub-index: dpr_faiss/title_first_paragraph
  searched content: title + first paragraph dense vectors
  input: one default DPR query embedding
- sub-index: dpr_faiss/title_entire_article
  searched content: title + entire article dense vectors
  input: one default DPR query embedding
- merge inside faiss_score: use the best normalized document score across the 4 FAISS sub-indexes

natural_questions_avg_faiss_score
- sub-index: dpr_faiss/title_first_sentence
  searched content: title + first sentence dense vectors
  input: natural question embeddings 1 to 5
- sub-index: dpr_faiss/title_first_two_sentences
  searched content: title + first two sentences dense vectors
  input: natural question embeddings 1 to 5
- sub-index: dpr_faiss/title_first_paragraph
  searched content: title + first paragraph dense vectors
  input: natural question embeddings 1 to 5
- sub-index: dpr_faiss/title_entire_article
  searched content: title + entire article dense vectors
  input: natural question embeddings 1 to 5
- merge inside natural_questions_avg_faiss_score:
  first use the best normalized document score across the 4 FAISS sub-indexes for each natural question
  then average across all 5 natural-question retrieval results

natural_questions_concat_faiss_score
- sub-index: dpr_faiss/title_first_sentence
  searched content: title + first sentence dense vectors
  input: one concatenated-natural-questions DPR embedding
- sub-index: dpr_faiss/title_first_two_sentences
  searched content: title + first two sentences dense vectors
  input: one concatenated-natural-questions DPR embedding
- sub-index: dpr_faiss/title_first_paragraph
  searched content: title + first paragraph dense vectors
  input: one concatenated-natural-questions DPR embedding
- sub-index: dpr_faiss/title_entire_article
  searched content: title + entire article dense vectors
  input: one concatenated-natural-questions DPR embedding
- merge inside natural_questions_concat_faiss_score: use the best normalized document score across the 4 FAISS sub-indexes
